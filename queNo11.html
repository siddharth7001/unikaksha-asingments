Programming Languages

Python

Year of Creation: 1991

Python is a high-level, general-purpose programming language known for its simplicity and readability.

JavaScript

Year of Creation: 1995

JavaScript is a scripting language primarily used for building interactive web pages and web applications.

C

Year of Creation: 1972

C is a low-level programming language known for its efficiency and close relationship with hardware.

Java

Year of Creation: 1995

Java is a versatile, object-oriented language widely used for developing desktop, web, and mobile applications.

Rust

Year of Creation: 2010

Rust is a systems programming language focused on safety, performance, and concurrency.

Swift

Year of Creation: 2014

Swift is a powerful and intuitive programming language developed by Apple for iOS, macOS, and more.

PHP

Year of Creation: 1995

PHP is a popular server-side scripting language used for web development and building dynamic websites.

Go

Year of Creation: 2009

Go, also known as Golang, is a statically typed language designed for simplicity, concurrency, and scalability.

C++

Year of Creation: 1983

C++ is an extension of the C language and is widely used for systems programming and game development.

Ruby

Year of Creation: 1995

Ruby is a dynamic, object-oriented language known for its elegant syntax and focus on developer happiness.
